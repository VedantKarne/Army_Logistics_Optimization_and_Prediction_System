## Setup Commands

```powershell
# Create and activate virtual environment
python -m venv generate_data_env
generate_data_env\Scripts\Activate.ps1

# Install dependencies
pip install -r Army_ML_Pipeline_and_Files\requirements.txt
```

## Key Output Files

| File | Generated By | Purpose |
|---|---|---|
| `vehicle_features.parquet` | `feature_engineering.py` | Feature matrix (5000 × ~59) |
| `models/xgb_best.pkl` | `train_health_model.py` | Trained XGBoost model |
| `models/lgbm_best.pkl` | `train_health_model.py` | Trained LightGBM model |
| `models/tabnet_best.zip` | `train_health_model.py` | Trained TabNet model |
| `models/ensemble_config.json` | `optimize_ensemble.py` | Champion strategy + weights |
| `reports/ensemble_roc_curve.png` | `evaluate_ensemble.py` | Multi-class ROC curves |
| `reports/shap_global.png` | `train_health_model.py` | SHAP feature importance |
| `health_scores` (DB table) | `run_inference.py` | Predictions for all vehicles |

## Health Status Classes

| Status | Risk | Action |

| 🟠 Poor | high | Withdraw from duty — repair within 48h |
| 🟡 Attention | medium | Schedule maintenance within 7 days |
| 🟢 Good | low | Proceed with next scheduled PM |
| ✅ Excellent | low | Continue normal operations |
